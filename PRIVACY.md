# Privacy Policy

Last updated: 2026-07-19

## Local Storage

Add a Line stores notebook data and embedded images locally on the user's Mac. It does not create an account, collect analytics, or send note text to a service operated by this project.

## Files

When a user opens or saves a file, Add a Line reads or writes only the file selected by that user. A pane can retain a local source-file path so that saving can write back to the same file.

## Web Images

When a user drags an image from a web page into a pane, Add a Line requests the selected image URL and embeds the downloaded image in the local notebook. The destination website receives a standard request for that image. The project does not receive the image, the note, or the URL.

## Changes

Substantive changes to this policy will be recorded in the project repository.
